# Mine Legends

## Vision
2D mining game for Yandex Games inspired by Gold Digger FRVR.

### Core Loop
1. Dig
2. Collect resources
3. Return to surface
4. Sell resources
5. Upgrade equipment
6. Dig deeper

### Progression
- Pickaxe upgrades
- Backpack upgrades
- Character upgrades
- Rare artifacts

### Monetization
- Rewarded ads: x2 coins
- Rewarded ads: treasure chest
- Battle Pass
- Cosmetic shop
- Remove ads purchase

### Leaderboards
- Deepest depth
- Weekly leaderboard
- Seasonal leaderboard

### Seasons
30-day seasons with cosmetic rewards.

### MVP
- 5 biomes
- 20 ores
- Account save
- Leaderboards
- Achievements
- Battle Pass
- Cosmetics
