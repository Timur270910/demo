# Mine Legends

## Vision
2D mining game for Yandex Games inspired by Gold Digger FRVR.

## Core Loop
1. Dig
2. Collect resources
3. Return to surface
4. Sell resources
5. Upgrade equipment
6. Dig deeper

## Biomes
### Surface (0-100m)
Dirt, Coal, Copper

### Stone Caves (100-300m)
Stone, Iron, Chests

### Golden Depths (300-700m)
Gold, Gems

### Crystal Caverns (700-1200m)
Crystals, Rare Artifacts

### Ancient Core (1200m+)
Legendary Resources

## Economy
Resources increase in value with depth.

Starter progression:
- Copper = 1
- Coal = 2
- Iron = 5
- Gold = 15
- Crystal = 50
- Ancient Relic = 250

## Upgrades
### Pickaxe
- Power
- Speed
- Critical Hit Chance
- Mining Radius

### Backpack
- Capacity levels 20 -> 40 -> 80 -> 160 -> 320

### Character
- Movement Speed
- Jump Height
- Magnet Range
- Lantern Strength

## Artifacts
Rare collectible items.

Examples:
- Ancient Crown
- Planet Core Fragment
- Legendary Drill
- Heart of Volcano
